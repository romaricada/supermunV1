Administration de la Plateforme de suivi des performances



Description du module : {{descTitle}}

Ce module offre la possibilité à l'administreur de gérer les niveau d'accès au données de l'interface d'administration.


Cela consiste à :
  • Créer les profils utilisateurs
  • Créer les comptes utilisateurs
  • Associer à chaque compte un profil
  • Activer/Désactiver un compte utilisateur
  • Changer le profil d'un compte utilisateur

Il permet également à l'utilisateur d'apporter certaines modifications à son compte.


Les actions possibles sont :
  • Demander une réinitilisation du mot de passe
  • Se conecter à la plateforme
  • Modifier ses informations personnelles
  • Modifier son mot de passe

Ce module offre la possibilité à l'utulisateur de gérer les performence. C'est à dir l'importation, la modification, la validation et la publication des données.


Cela consiste à :
  • Exporter un model de fichier
  • Importer un fichier renseigné
  • Modifier des données
  • Valider et publier des données

Il permet également à l'utilisateur exporter des données sous plusieurs format(XLS, CSV).


Ce module offre la possibilité à l'utulisateur de gérer les Indicateur.


Cela consiste à :

Gestion des groupes de domaines

  • Créer un nouveau groupe de domaine
  • Modifier un groupe de domaine
  • Supprimer un groupe de domaine

Gestion des domaines

  • Créer un nouveau domaine d'indicateur
  • Modifier un domaine d'indicateur
  • Supprimer un domaine d'indicateur

Gestion des indicateur

  • Créer un nouveau d'indicateur
  • Modifier un indicateur
  • Supprimer un indicateur
  • gerer les valeurs modalités

Ce module offre la possibilité à l'administreur de repartir au portail


Le portail permet de :
  • Visualiser de la carte du Burkina Faso avec les communes
  • Consulter de la performance d'une commune pour un indicateur donnée à partir de la carte
  • Naviguer vers la page des données ou vers la page des posters
  • Acceder au site du open data pour consulter les données : OpenData

En plus le portail offre à un internaute d'énormes fonctionnalités


Les actions possibles sont :
  • Consulatation des données des performances municipales
  • Téléchargement des données des performances municipales
  • Consultation et téléchargement des posters
  • Consultation des informations et des publication
  • Utilisation des APIs pour récupérer des données publiées sur le portail
